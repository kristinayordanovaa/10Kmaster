# 10K to Mastery - Landing Page

A modern, conversion-optimized landing page for a time tracking app focused on helping users achieve mastery through the 10,000-hour rule.

## Overview

**10K to Mastery** is a time tracking application that helps users track their journey to expertise. This landing page is designed to convert visitors into users through compelling copy, social proof, and clear value propositions.

## Features

### Landing Page Highlights

- **Hero Section** with compelling headline and social proof statistics
- **Interactive Dashboard Mockup** showcasing the app's visual progress tracking
- **Feature Showcase** highlighting 6 key benefits
- **3-Step Process** explaining how simple it is to get started
- **Social Proof** with user testimonials and ratings
- **Pricing Section** with 3 tiers (Free, Premium, Lifetime)
- **Multiple CTAs** strategically placed throughout the page
- **Fully Responsive** design that works on all devices
- **Smooth Animations** and hover effects for enhanced user experience

### Design Elements

- Modern gradient color scheme (Purple/Indigo)
- Clean typography using Inter font family
- Smooth scrolling navigation
- Animated elements on scroll
- Professional card-based layouts
- Mobile-first responsive design

## Getting Started

### Installation

1. Clone or download the project files
2. No build process or dependencies required

### Usage

Simply open `index.html` in any modern web browser:

```bash
# Option 1: Direct file open
open index.html

# Option 2: Using Python's built-in server
python -m http.server 8000

# Option 3: Using Node.js http-server
npx http-server
```

Then navigate to `http://localhost:8000` in your browser.
